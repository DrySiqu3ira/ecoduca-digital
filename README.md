# Ecoduca Digital

Pequeno site educativo sobre descarte consciente de eletrônicos.

## Estrutura

- `index.html` — página inicial
- `descarte.html` — pontos de descarte
- `quiz.html` / `resultado.html` — quiz e resultado
- `assets/css/style.css` — estilos
- `assets/js/script.js` — scripts

## Como executar (local)

1. Abra `index.html` no navegador; ou
2. Rode um servidor simples (recomendado):

- Com Python 3: `python -m http.server 8000`
- Acesse: `http://localhost:8000`

## Como contribuir

- Faça um fork / branch, edite e abra um pull request.
- Mantenha CSS em `assets/css/style.css` e scripts em `assets/js/script.js`.
